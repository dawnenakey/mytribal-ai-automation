Since Google Voice costs $10/month, **TextNow is your best free option** for Amazon Associates verification.

## Step 1: Sign Up for TextNow (Free)

1. **Go to:** https://www.textnow.com
2. **Click:** "Sign Up" or "Get Free Number"
3. **Choose Sign-Up Method:**
   - Email sign-up (recommended)
   - Or Google/Apple account
4. **Create Account:**
   - Enter your email (can use admin@mytribal.ai)
   - Create password
   - Verify email if required
5. **Get Your Free Number:**
   - Choose area code (select one in your region)
   - Pick a number from available options
   - Number is yours free forever

## Step 2: Access TextNow

### Option A: Web Browser (Easiest)
- Go to: https://www.textnow.com
- Log in
- Use web interface to check messages

### Option B: Mobile App
- Download TextNow app (iOS/Android)
- Log in with same account
- Check messages on phone

### Option C: Desktop App
- Download desktop app if preferred

## Step 3: Use for Amazon Associates

1. **Sign Up for Amazon Associates:**
   - Go to: https://affiliate-program.amazon.com
   - Enter admin@mytribal.ai as email
   - Enter your TextNow phone number when prompted

2. **Receive Verification SMS:**
   - Amazon will send SMS code to TextNow number
   - Check TextNow (web or app) for the message
   - Code should appear within 1-2 minutes

3. **Enter Verification Code:**
   - Copy the code from TextNow
   - Paste into Amazon Associates form
   - Complete signup

## Step 4: Keep Access to TextNow

**Important:** Keep TextNow accessible because:
- Amazon may send verification codes again
- You might need to verify on future logins
- Keep the app/web bookmark handy
