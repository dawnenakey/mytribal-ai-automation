# How to Access Site Editor for Navigation

## Step 1: Log Into WordPress

1. Go to: https://mytribal.ai/wp-admin
2. Enter your WordPress username and password
3. Click "Log In"

## Step 2: Access Site Editor

After logging in, you have **two options**:

### Option A: Direct Link (Easiest)
Once logged in, go directly to:
**https://mytribal.ai/wp-admin/site-editor.php**

### Option B: Via Admin Menu
1. In WordPress admin, click **"Appearance"** in the left sidebar
2. Look for **"Editor"** or **"Theme Editor"** 
3. Click it - this should take you to Site Editor

**Note:** If you don't see "Editor" under Appearance, your hosting might restrict it. Use Option C instead.

### Option C: Via Appearance → Customize (Alternative)
If Site Editor isn't available:
1. Go to: **Appearance → Customize**
2. Look for **"Menus"** or **"Navigation"** section
3. Add your menu there

## Step 3: Once in Site Editor

1. **Click "Templates"** in the left sidebar
2. **Find "Header"** in the template list
3. **Click on "Header"** to edit it
4. **Click "Edit"** button if it appears

## Step 4: Add Navigation Block

1. Click the **"+"** button (usually top left or in the editor)
2. Search for **"Navigation"**
3. Click the **"Navigation"** block
4. In right sidebar, select your menu from dropdown

## Step 5: Save

Click **"Save"** button (top right)

---

## If Site Editor Doesn't Work

Try the **Customizer** method instead:

1. Go to: **Appearance → Customize**
2. Look for **"Menus"** section
3. Click **"Add a Menu"** if needed
4. Select your menu
5. Check the **"Primary Menu"** or **"Main Menu"** location
6. Click **"Publish"**
