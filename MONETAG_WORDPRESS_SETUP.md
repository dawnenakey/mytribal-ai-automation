# Monetag WordPress Plugin Setup Guide

## ✅ Plugin Status
The Monetag plugin shows "✔ Installed!" - now you need to activate and configure it.

## Step 1: Activate the Plugin

1. **Go to Installed Plugins:**
   - In WordPress admin, click: `Plugins` → `Installed Plugins` (left sidebar)

2. **Find Monetag:**
   - Look for "Monetag Official Plugin"
   - Click `Activate` if it's not already active

3. **Verify Activation:**
   - Once activated, you should see "Monetag" in your WordPress menu

## Step 2: Get Your Monetag Ad Code

After signing up at publishers.monetag.com, you need your ad code:

1. **Log into Monetag Dashboard:**
   - Go to: https://publishers.monetag.com
   - Log in with your account

2. **Create Ad Units:**
   - Navigate to "Ad Units" or "Sites" section
   - Add your site: `mytribal.ai`
   - Create ad units (banner sizes like 728x90, 300x250, etc.)
   - Copy the ad code/ID for each unit

3. **Common Ad Sizes:**
   - **728x90** (Leaderboard - top of posts)
   - **300x250** (Medium Rectangle - sidebar)
   - **300x600** (Half Page - sidebar)
   - **Native Ads** (in-content)

## Step 3: Configure Monetag Plugin

1. **Access Monetag Settings:**
   - In WordPress admin, look for `Monetag` in the left sidebar
   - Click on it to open settings

2. **Add Your Site/Account:**
   - Enter your Monetag Site ID or Account ID
   - This connects your WordPress site to your Monetag account

3. **Configure Ad Units:**
   - The plugin may have a simple interface to:
     - Add ad unit IDs
     - Set placement locations
     - Configure auto-placement options

4. **Alternative: Manual Code Insertion**
   - If the plugin doesn't auto-place ads, you may need to:
     - Use widget areas for sidebar ads
     - Use shortcodes for in-content ads
     - Or manually insert codes in theme files

## Step 4: Test Ad Display

1. **Visit Your Site:**
   - Go to: https://mytribal.ai
   - Check if ads are displaying

2. **Check Different Pages:**
   - Homepage
   - Single post page
   - Archive pages

3. **Disable Ad Blocker:**
   - Make sure your browser ad blocker is off for testing

## Step 5: Verify in Monetag Dashboard

1. **Check Monetag Dashboard:**
   - Log into: https://publishers.monetag.com
   - Look for impressions/views
   - Should show activity within 24-48 hours

## Troubleshooting

### Ads Not Showing?
- ✅ Make sure plugin is activated
- ✅ Check your Site ID/Account ID is correct
- ✅ Clear WordPress cache
- ✅ Clear browser cache
- ✅ Disable ad blocker
- ✅ Wait 24-48 hours (can take time to propagate)

### Plugin Not Working?
- ✅ Check WordPress version compatibility
- ✅ Try deactivating and reactivating
- ✅ Check for plugin conflicts
- ✅ Contact Monetag support

### Need Help?
- **Monetag Support:** support@monetag.com
